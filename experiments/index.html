MNIST Dataset (Dataset of Handwritten Digits) - Dimensionality Reduction Methods (PCA, t-SNE, UMAP)
The goal is to show some images from MNIST dataset and show the projection of the images in the dataset to 2D coordinates via Principal Component Analysis (PCA), t-distributed Stochastic Neighbour Embedding (t-SNE), and Uniform Manifold Approximation and Projection (UMAP). These projections are used to show the way each image on the dataset is spread based on its label in different dimensions (here in 2D dimension). Ideally, the images belonging to one class (e.g. image with label 5) should create a cluster and images with different labels (e.g. label 5 vs. label 6) should belong to different clusters.
